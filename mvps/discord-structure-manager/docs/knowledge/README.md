# Knowledge

このMVPの進行中に得られた知見を蓄積します。

## 何を残すか

- 調査して分かったこと
- 失敗した試行と原因
- 採用した実装方針と理由
- 外部サービス連携で詰まった点
- 次に同じ作業をするときの注意点

## 書き方

1テーマ1ファイルを基本にし、ファイル名は `YYYY-MM-DD-topic.md` または `topic.md` にします。

```md
# Topic

## Context

TBD

## Findings

TBD

## Tried

TBD

## Decision

TBD

## Next Time

TBD
```
