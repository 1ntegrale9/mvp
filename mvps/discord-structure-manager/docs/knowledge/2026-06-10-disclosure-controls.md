# Disclosure Controls

## Context

カテゴリ、スレッド内包チャンネル、ロール一覧は初期状態で閉じるため、開閉できる場所を迷わないUIにする必要がある。

## Decision

- 開閉ボタンは、チェックボックスやドラッグハンドルより左に置く。
- ロール一覧の開閉ボタンも、パネル右側のツールバーではなく見出しの一番左に置く。
- `+` / `-` は追加・削除操作にも見えるため、閉じている状態は `▶`、開いている状態は `▼` のdisclosure iconで表す。
- アイコンだけのボタンでも、`aria-label` と `title` には「開く / 閉じる」を明記する。

## Notes

選択チェックボックスは一括反映対象の指定、開閉ボタンは表示密度の操作なので、左から「開閉 -> 選択 -> 並び替え」の順に並べると役割が混ざりにくい。
