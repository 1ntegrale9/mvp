# Category Sync Notes

## Context

Discordのチャンネル権限には、カテゴリ権限と同期しているチャンネルと、個別権限を持つチャンネルがある。管理画面MVPでもこの差が見えないと、権限変更の反映先が分かりづらくなる。

## Decision

- 子チャンネルは `synced: true | false` を持つ。
- 同期ONのチャンネルは、権限エディタ上では親カテゴリの権限を表示する。
- 同期ONのチャンネルへ権限を反映する場合、実際の反映先は親カテゴリにする。
- 同期OFFのチャンネルへ権限を反映する場合、実際の反映先はそのチャンネルにする。
- 同期ONへ戻すときは、そのチャンネル固有の上書きを空にする。

## UX

- チャンネル行に `カテゴリ同期` / `個別権限` の状態バッジを表示する。
- 各チャンネル行に `同期ON` / `同期OFF` ボタンを常時表示する。
- 移動操作パネルに、選択チャンネルをまとめて同期ON/OFFにするボタンを置く。
- 権限エディタ上部に、現在の選択が同期ON、同期OFF、混在、カテゴリ選択中のどれかを説明する。
